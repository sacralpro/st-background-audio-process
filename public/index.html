Sacral Track Audio Processor

Status

This is a background service for audio processing. There is no user interface.

Service status: Running

About

This service automatically processes WAV audio files in Sacral Track:

  • Converts WAV to MP3
  • Creates HLS segments for streaming
  • Generates m3u8 playlists
  • Updates database records
